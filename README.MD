# Realistic Starfield — Lively Wallpaper

An interactive, performance-optimized HTML5 Canvas space simulation designed as a dynamic live wallpaper for **Lively Wallpaper**. It features drifting nebulae, twinkling stars, cosmic orbs, and occasional shooting stars.

Developed by **Raspberrykitty1**.

## 🚀 Features

* **Parallax Star Layers:** Three distinct layers of stars moving at independent depths and speeds to create an immersive 3D parallax effect.
* **Twinkling & Drifting:** Stars dynamically shift their transparency, while custom radial-gradient nebulae and cosmic orbs gently drift across your desktop background.
* **Object-Pooled Shooting Stars:** Smoothly animated shooting stars that utilize an object pool to prevent performance stutters and heavy garbage collection while your desktop is running.
* **Fully Responsive:** Automatically scales and repositions all cosmic elements flawlessly whenever your screen resolution or desktop orientation changes.

## 📁 Project Structure

* `index.html` — The main entry point and canvas markup for the wallpaper.
* `script.js` — Core simulation logic, animation loops, and particle management.
* `style.css` — Minimal, full-bleed canvas styling to ensure a borderless desktop fit.
* `LivelyInfo.json` — Metadata required by Lively Wallpaper to recognize the author, type, and entry point.

## 📦 How to Install in Lively Wallpaper

1. **Download the ZIP archive** containing all the project files.
2. Open **Lively Wallpaper**.
3. Click the **Plus (+) button** (Add Wallpaper).
4. Select **Open File**.
5. Select the downloaded **ZIP archive**.
6. **Activate it** to set it as your dynamic background!
