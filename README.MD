# Realistic Starfield — Lively Wallpaper (v4)

An interactive, performance-optimized HTML5 Canvas space simulation designed as a dynamic live wallpaper for **Lively Wallpaper**. Version 4 builds upon previous iterations by introducing complete background music controls and a massive selection of cosmic text color customizations for the performance benchmark overlay.

Developed by **Raspberrykitty1**.

## 🚀 Features

* **Parallax Star Layers:** Three distinct layers of stars moving at independent depths and speeds to create an immersive 3D parallax effect.
* **Advanced Live UI Properties:** Fully integrated with Lively Wallpaper's configuration engine, giving you deep control over audio states, simulation density, and diagnostic overlay cosmetics.
* **Customizable Performance Benchmarking:** Built-in performance tracking (`benchmark.js`) calculating live, average, minimum, and maximum FPS alongside render processing latency. 
* **Ambient Audio Support with Mute Toggle:** Background track execution featuring an absolute mute toggle control to easily switch audio off or on from your Lively interface. The background music is a compressed, audio-only extraction of the track from [this video](https://youtu.be/gCWaRhNUvfc).
* **Object-Pooled Shooting Stars:** Smoothly animated shooting stars utilizing an object pool to mitigate runtime memory allocations and performance hiccups.

## 📁 Project Structure

* `index.html` — The main entry point, visual canvas layout, and background audio container.
* `script.js` — Core simulation logic, animation rendering loop, audio management, and particle updates.
* `benchmark.js` — Diagnostics module housing the `BenchmarkOverlay` class, updated to dynamically calculate multi-corner bounds and theme colors.
* `style.css` — Full-bleed layout rules keeping components seamlessly aligned at absolute boundaries.
* `space-ambient.opus` — The compressed background audio file used for ambient space tracking.
* `LivelyInfo.json` — Operational metadata enabling environment configuration parsing inside Lively.
* `LivelyProperties.json` — Enhanced settings schema handling interactive audio checkboxes and expanded color dropdown menus.

## 🛠️ Customizable Settings

The following user configurations can be adjusted in real time through the Lively user interface:

* **Mute Background Music (Checkbox):** Easily mute or play the ambient space soundtrack dynamically.
* **Show Performance Overlay (Checkbox):** Toggle the monospaced text diagnostic overlay on or off.
* **Overlay Position (Dropdown):** Snap the performance overlay box to any corner of your screen (**Top-Right**, **Top-Left**, **Bottom-Right**, or **Bottom-Left**).
* **Overlay Text Color (Dropdown):** A massive selection of 18 diagnostic text colors to perfectly match your desktop theme, including standard staples (**Red**, **Lime Green**, **Cyan**, **White**, **Yellow**) as well as premium cyber and pastel hues (**Magenta**, **Orange**, **Deep Pink**, **Hot Pink**, **Gold**, **Chartreuse**, **Spring Green**, **Deep Sky Blue**, **Lavender**, **Plum**, **Aquamarine**, **Medium Purple**, and **Crimson**).
* **Overlay Background (Dropdown):** Adjust the visibility style of the benchmark background box (**Semi-Transparent**, **Opaque Black**, or **Invisible**).
* **Star Density (Slider):** Scales active star clusters smoothly across runtime arrays. Supports adjustments ranging from 100 to 10,000 active instances.

## 📦 How to Install in Lively Wallpaper

1. **Download the ZIP archive** containing all the version 4 project assets.
2. Open **Lively Wallpaper**.
3. Click the **Plus (+) button** (Add Wallpaper).
4. Select **Open File**.
5. Select the downloaded **ZIP archive**.
6. **Activate it** to apply the updated space environment!
7. Right-click the thumbnail in your Lively collection library and select **Customize** to access your brand new audio and expanded color palette controls.
